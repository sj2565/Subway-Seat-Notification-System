# Subway-Seat-Notification-System
수정하기

향후 개선할 점
1. C -> Node(서버)로 보낼 때 단순 버퍼 형식이 아니라 UART나 TCP/UDP로 활용

4. 메모리 관리를 위해 직접 할당
